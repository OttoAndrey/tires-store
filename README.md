# tires-store
Example of Django project

```bash
$ cd tires-store
```

```bash
$ python3.7 -m venv .virtualenv
```

```bash
$ source .virtualenv/bin/activate
```

```bash
$ pip install -r requirements/development.txt
```

```bash
$ docker-compose up -d postgres
```

```bash
$ ./manage.py migrate
```

```bash
$ ./manage.py createsuperuser
```

```bash
$ ./manage.py runserver
```

Get fun!
